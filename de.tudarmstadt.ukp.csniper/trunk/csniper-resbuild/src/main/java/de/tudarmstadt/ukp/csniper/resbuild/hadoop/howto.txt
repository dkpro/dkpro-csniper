how to for future me.

1. run maven build, goals: dependency:copy-dependencies package
2. copy jars and /dependencies to headnode
3. copy BncHadoopCorpusBuilder.properties, run.sh to headnode, directly to user home dir
4. create exclusions.txt on headnode user home dir if needed
5. ssh to headnode, start the hadoop job with nohup:
	screen ./run.sh de.tudarmstadt.ukp.csniper.resbuild.hadoop.BncHadoopCorpusBuilder input output &
	ctrl +A, D // zum detachen
	screen -r // zum anmelden
