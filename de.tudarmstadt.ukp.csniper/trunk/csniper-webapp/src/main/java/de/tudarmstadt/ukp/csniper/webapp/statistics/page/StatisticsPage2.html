Beware that calculating the query statistics may take a very long time.

Explanation

  • A hit counts as a true positive (TP) a relevant majority of annotators say that is a correct result. A majority is relevant it is above the threshold compared to the total number of users .
  • The precision is calculated using the true positives (TP) and false positives (FP) with unknown (UNK) items counting towards the false positives. For the estimated precision the unknown items are not taken into account.
  • Complete is the percentage of true positives (TP) and false positives (FP) compared to the total number of items. This indicates how far the annotation task for this query has already progressed.
  • Correct and wrong are total number of correct and wrong votes that all users have placed for items in this query.
  • Kappa is calculated using Fleiss' Kappa.
